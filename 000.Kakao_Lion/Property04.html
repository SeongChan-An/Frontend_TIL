z-index

예제(1) z-index를 아무것도 지정해 주지 않으면 기본으로 0으로 적용되고, dom구조 하단에 위치 할 수록 우선순위는 올라갑니다. (A < B < C)

예제(2)에서는 B를 C보다 위에 놓은 경우입니다. B에만 z-index:1 을 주면 나머지 요소들은 0 이니까 최상단으로 올라오게 되는 것 입니다. (A < C < B)


https://developer.mozilla.org/ko/docs/Web/CSS/Understanding_z-index/Adding_z-index

예제(1)
A (z-index: 0)
B (z-index: 0)
C (z-index: 0)
예제(2)
A (z-index: 0)
B (z-index: 1)
C (z-index: 0)
